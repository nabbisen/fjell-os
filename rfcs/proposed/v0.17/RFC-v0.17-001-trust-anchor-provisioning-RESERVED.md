# RFC-v0.17-001: Trust Anchor Provisioning and Manufacturing Flow (RESERVED)

**Status:** Reserved (not yet drafted)
**Milestone:** v0.17
**Origin:** Deferred from RFC-v0.16-005 (architect review H-02).

This number is reserved for the trust-anchor provisioning RFC — how a
`TrustAnchorRoot` is provisioned onto a freshly manufactured node. It is a
v1.0 limitation acknowledged in the v0.16 release notes and is **not** part
of the Verus adoption program (RFC-v0.17-002 … 006).

Drafting is pending. This placeholder exists so the v0.17 RFC numbering is
unambiguous and the prior reference from RFC-v0.16-005 resolves.
