# What is Fjell?

> **Fjell OS is a verifiable, capability-based operating system for high-assurance edge and fleet nodes where every authority, update, recovery action, and runtime state must be explainable.**

See [v1.0 Direction and Identity](../identity/v1-direction.md) for the full definition, archetypes, and invariants.
