# Three-Node Fleet Tutorial

Demonstrates the complete fleet flow: build → enrol → deploy bundle → verify.

See [`examples/three-node-fleet/README.md`](../../examples/three-node-fleet/README.md) for the full step-by-step walkthrough.

```bash
cargo xtask fleet-demo deploy
# Runs all steps and reports PASS or FAIL
```
